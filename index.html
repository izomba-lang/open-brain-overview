Скиллы

Open Brain теперь умеет не только помнить, но и действовать

Скилл — это готовый рецепт для задачи.
Говоришь «подготовь бриф к встрече» — система знает, что делать:
найти заметки, посмотреть участников, собрать задачи, сформировать документ.

Не нужно каждый раз объяснять AI, как действовать — скилл уже содержит пошаговую инструкцию.

Скиллы в деле реальные сценарии

Подготовка к встрече

Ты: «Подготовь меня к завтрашнему созвону с Дмитрием»
Скилл meeting-brief автоматически:
• Находит все прошлые обсуждения с Дмитрием
• Поднимает открытые задачи по общим проектам
• Собирает бриф: цель, темы, контекст
Результат: готовый бриф за 30 секунд вместо 20 минут

Фоллоу-ап после встречи

Ты: «Напиши итоги встречи с продуктовой командой»
Скилл follow-up автоматически:
• Достаёт заметки из Granola по этой встрече
• Извлекает договорённости, ответственных, дедлайны
• Пишет письмо в твоём стиле
Результат: готовый фоллоу-ап, осталось только отправить

Делегирование задачи

Ты: «Поручи Лене подготовить данные по Беларуси»
Скилл delegate-task автоматически:
• Находит контекст проекта и роль Лены
• Формулирует задачу: что, зачем, к какому сроку
• Адаптирует тон под отношения с человеком
Результат: чёткая формулировка, а не «ну ты поняла»

Сводка для руководства

Ты: «Подготовь сводку за месяц для директора»
Скилл weekly-report автоматически:
• Собирает все завершённые задачи и решения
• Группирует по проектам, выделяет достижения
• Формирует executive summary с ключевыми цифрами
Результат: отчёт, на который раньше уходил час

Откуда брать скиллы

1

Создать самому

Описываешь простым языком: что скилл делает, по каким словам его вызывать, какие шаги выполнять. Всё — скилл готов к работе.

Пример: «Создай скилл «onboarding» — когда приходит новый сотрудник, собери всю информацию о его роли, подготовь план первой недели и список людей для знакомства»
2

Импортировать из интернета

Нашёл хорошее описание процесса или роли? Одна команда — и Open Brain автоматически превращает его в скилл: парсит, адаптирует под твои данные, сохраняет.

Пример: нашёл на GitHub описание «Product Manager Agent» — импортировал, и теперь Open Brain умеет помогать с приоритизацией фич и дорожной картой.
3

Скилл появляется сам

Когда создаёшь задачу, Open Brain автоматически подбирает лучший скилл. Задача «написать письмо Алексею по итогам ревью» сразу получает бейдж подходящего скилла.

Это работает как навигатор: ты говоришь куда ехать, а система сама строит маршрут.

Сейчас в системе 9 скиллов: подготовка к встречам, фоллоу-апы, отчёты, делегирование, фиксация решений, черновики писем, сводки для руководства и другие. Библиотека растёт.
